#Hulu2.0-Clone


## Tech Stack

**Client:** NextJs, TailwindCSS, Axios, json-server, vue-router

**Server:** Node, Express




## Installation

Install hulu2.0 with npm

```bash
  npm install
  cd hulu2.0
```
